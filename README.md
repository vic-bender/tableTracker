# Victor's Warhammer tableTracker
This is a simple app that I am making (currently a work-in-progress) to track Warhammer 40,000 games that I (and my friends) play over the board. This app was made by me because I can't be bothered to use the other apps already out there, they're too complicated for no reason. The IDE I used was VSCode.
